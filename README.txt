To create sphinx documents do:
i) pip install sphinx
   pip install sphinxcontrib.bibtex
   pip install sphinx_rtd_theme

ii) sphinx-build -b html <from_DIR> <to_DIR>
e.g. sphinx-build -b html ./source ./test

Re-Structure-Text guidelines at: https://www.sphinx-doc.org/en/master/usage/restructuredtext/index.html
